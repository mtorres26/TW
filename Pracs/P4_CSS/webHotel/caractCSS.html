Características del CSS adaptable

Resoluciones entre 600px y 1024px

  • Los elementos "h2" no deben de salirse de la caja de contenidos al ir minimizando progresivamente la ventana.
  • Análogamente lo mismo debe de suceder con las imágenes.
  • El "aside" no debe superponerse a los demás elementos.
  • Los elementos del "nav" no deben superponerse entre ellos mismos al ir minimizando progresivamente la ventana.

Resoluciones menores de 600px

  • Los elementos con imágenes se disponen en la misma fila todos para ocupar menos ancho.
  • El "nav" se dispone en filas para ocupar menos espacio y que sea igual de legible.
  • Desaparece el elemento "aside"
